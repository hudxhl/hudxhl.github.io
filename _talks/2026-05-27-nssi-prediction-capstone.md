## Overview

Presented a machine learning capstone project focused on predicting recent nonsuicidal self-injury using clinical, psychosocial, and behavioral health data. The project translated a sensitive behavioral health research question into a supervised classification problem, with the goal of evaluating whether recent NSSI could be predicted from participant-level indicators in a way that was both technically rigorous and interpretable.

The workflow included outcome engineering, data cleaning, missingness handling, train-validation-test splitting, model comparison, and performance evaluation. I compared multiple classification approaches, including K-nearest neighbors, random forest, and regularized logistic regression, and evaluated model performance using metrics such as ROC AUC and feature importance.

The presentation emphasized both predictive performance and responsible interpretation. Because the outcome was clinically sensitive, I focused on how model results could be used to generate insight rather than replace human judgment. I presented findings, limitations, and future directions to key stakeholders, including opportunities to improve model calibration, examine subgroup performance, and develop more interpretable risk-related features.
